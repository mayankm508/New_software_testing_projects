# IMDb Website Automation Project

## Overview
This project automates the testing of the IMDb website using Selenium and Pytest. It includes features like login validation, movie search, and API testing to ensure the website's core functionalities work as expected.

## Features
- Automated login testing
- Movie search functionality testing
- Modular framework with utilities for reusability
- Detailed HTML reports for test results

## Requirements
- Python 3.8 or higher
- Google Chrome browser
- ChromeDriver compatible with your Chrome version
